# 8.1 Install

Using npm:

```shell
npm install axios
```

Using bower:

```shell
bower install axios
```

Using yarn:

```shell
yarn add axios
```

Using jsDelivr CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
```

Using unpkg CDN:

```html
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
```
